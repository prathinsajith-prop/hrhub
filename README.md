# HRHub.ae — UAE HR & PRO Management Platform

Production-grade multi-tenant SaaS HR platform for UAE businesses.

## Quick Start
npm install && npm run dev

Demo: sarah.j@hrhub.ae / password123

## Stack
React 19 + TypeScript, Vite 8, Tailwind CSS 4, Radix UI, Zustand, TanStack Table, React Hook Form + Zod, Recharts
